COMPONENTES DE UNA RED

  • Servidor:este ejecuta el sistema operativo de red y ofrece los servicios de red a las estaciones de trabajo.
  • Estaciones de trabajo:Estaciones de Trabajo: Cuando una computadora se conecta a una red, la primera se convierte en un nodo de la última y se puede tratar como una estación de trabajo o cliente. Las estaciones de trabajos pueden ser computadoras personales:
    • con el DOS,
    • Macintosh,
    • Unix,
    • OS/2 o
    • estaciones de trabajos sin discos.
  • Tarjetas o Placas de Interfaz de Red
    1. Ethernet
    2. ArcNet
    3. Token Ring
